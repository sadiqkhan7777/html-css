JavaScript Interview Questions

Compulsory Question

Q1. What are the different data types present in JavaScript?

Jawab: JavaScript mein kayi data types hain:

  • String (Misal ke taur par, 'hello World', "hello World-2")
  • Number (Misal ke taur par, 25)
  • Boolean (Misal ke taur par, true, false)
  • Null (Misal ke taur par, null, jo kisi bhi value ko represent nahi karta)
  • Undefined (Misal ke taur par, aik variable jo declare kiya gaya hai lekin usay koi value assign nahi hui)
  • BigInt (Misal ke taur par, 0922992929292992)
  • Object (Misal ke taur par, const car = {type:"Fiat", model:"500", color:"white"})
  • Array (Misal ke taur par, const cars = ["Buggati", "Maclaren", "koenigsegg"])

Q2 What are the types of Pop-up boxes available in JavaScript?

Jawab: JavaScript mein teen tarah ke pop-up boxes hote hain:

  • Alert Box

    • User ko ek message dikhata hai (alert("Hello"))

  • Confirm Box

    • User se aik sawaal karta hai aur true ya false return karta hai (confirm("Kya aap sure hain?"))

  • Prompt Box

    • User se input maangta hai aur user input value return karta hai (prompt("Apni umar darj karein:"))

Q3. What is JavaScript?

Jawab: JavaScript aik scripting language hai jo webpages ko interactive banane ke liye istemal hoti hai. Yeh aapko HTML, CSS ko manipulate karne aur webpages ke behavior ko control karne ki ijazat deti hai.

Q4. What is ES6?

Jawab: ES6 yaani ECMAScript 6 ya ECMAScript 2015 aik nayi version hai jo JavaScript mein kai mufeed features laya taake coding ko aasaan aur zyada reliable banaya ja sake. JavaScript ke naye version mein let aur const introduce kiya gaya taake variables ko zyada controlled tareeke se declare kiya ja sake, purane var ki jagah. let aapko aik variable ki value ko badalne ki ijazat deta hai aur const un values ke liye hota hai jo badalni nahi chahiyein.

Q5. What is a Variable?

Jawab: Variable aik container hota hai jo programming languages mein data values store karne ke liye hota hai. JavaScript mein, aap aik variable ko var, let, aur const se declare kar sakte hain. Variable mukhtalif qisam ka data store kar sakta hai jaise ke strings, numbers, objects, arrays aur zyada.
